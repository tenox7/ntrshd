# NT RSHD - Windows NT RSH Daemon

- Intended for use with Claude Code etc.
- Insanely dangerously insecure, meant to be used only in secure LAN environments.
- This is NOT a Windows service. -
- You run it from your project folder on NT.
- It's also a good idea to mount the same directory so Claude etc can edit files locally and run nmake remotely.
